This Person Doesnt Exist - DCGAN

This is a deep learning project that uses DCGAN to create faces of people that don't exist would involve training a generator and a discriminator network. The generator network would be trained to produce realistic-looking faces of people, while the discriminator network would be trained to distinguish between real and fake faces.The final output of this project would be a generator network capable of generating new faces of people that don't exist, with each face having unique features and characteristics.
